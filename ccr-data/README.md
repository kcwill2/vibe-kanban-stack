# CCR (Claude Code Router) config

Copy `config.example.json` to `config.json` and fill in your API keys (or use `$VAR` placeholders and set env vars in `.env`).

- `config.json` is gitignored (contains secrets).
- See [DOCKER_SETUP.md](../DOCKER_SETUP.md) for provider setup (OpenRouter, Gemini, LM Studio).
